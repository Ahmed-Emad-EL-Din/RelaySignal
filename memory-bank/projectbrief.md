# RelaySignal Project Brief

## Core Definition
RelaySignal is an advanced task management and notification system designed to deliver reliable reminders regardless of internet connectivity, browser status, or device state.

## Primary Purpose
Solve the critical problem of missed task deadlines by delivering notifications even when:
- There is no internet connection
- The browser is completely closed
- The computer has been restarted
- The application tab is not open

## Core Requirements
✅ Default 5 hour reminder before all task deadlines
✅ User configurable custom reminder times
✅ Support for MULTIPLE reminders per single task
✅ 100% offline operation capability
✅ Native browser integration without external dependencies
✅ Fully responsive modern user interface

## Target Platforms
- Desktop Chrome / Edge / Chromium browsers
- Mobile Chrome for Android
- PWA installable application

## Deployment Target
Netlify Edge Functions + CDN hosting
